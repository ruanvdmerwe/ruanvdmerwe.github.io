This site is a small reset. The previous version had been quietly out of date for
a while, and the templating it was built on had become more maintenance than it
was worth.

The new version is deliberately quiet. Serif headings, generous whitespace, a
single column of text. The point is to make it easy to **write** &mdash; about
representation geometry, continual learning, multilingual speech, and what it
actually takes to ship research-driven AI into production.

A few notes on what lives where:

- **[Papers](/papers)** &mdash; peer-reviewed publications, each with PDF / arXiv /
  code links and a BibTeX entry you can copy.
- **[Projects](/projects)** &mdash; product and open-source work, including
  ByteFuse, Maski, and the code that accompanies the papers.
- **[Talks](/talks)** &mdash; conference talks, workshops, and media appearances.
- **[CV](/cv)** &mdash; the long-form version, with a print-friendly layout.

If you want to follow along, there&rsquo;s an [RSS feed](/rss.xml).
